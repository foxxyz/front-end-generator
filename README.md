Front-end Generator
===================

Zero-installation generation script for the [Vite & Vue 3 Front-End Starter](https://github.com/foxxyz/front-end-starter).

Requirements
------------

 * Node 12+
 * git

Usage
-----

```
npx generate-front-end
```

For all options, see:
```
npx generate-front-end --help
```
